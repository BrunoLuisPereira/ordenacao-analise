Análise de Algoritmos de Ordenação

Projeto desenvolvido para a disciplina, com o objetivo de comparar o desempenho de algoritmos de ordenação na prática, utilizando métricas reais, logs e observabilidade.

 Objetivo

Implementar e analisar algoritmos de ordenação, comparando:

- Tempo de execução
- Crescimento de complexidade (Big-O)
- Comportamento em diferentes tamanhos de entrada

Algoritmos implementados

- 🔹 Insertion Sort — O(n²)
- 🔹 Bubble Sort — O(n²)
- 🔹 Merge Sort — O(n log n)
